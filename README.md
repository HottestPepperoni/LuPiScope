DIY digital inspection tool using a Raspberry Pi HQ camera, C-mount lens (e.g. Tamron 23FM25SP), a Raspberry Pi 5 (a Pi 4 also works) and a 3D-printed housing.

Designed for PCB inspection, soldering, and magnification tasks. Features a height-adjustable stand and modular design. Ideal for makers and electronics enthusiasts.

## Repository layout

- `docs/parts/` — Bill of Materials, parts list and supplier links
- `hardware/3d/` — 3D models, CAD sources and printing notes
- `firmware/` — Raspberry Pi setup, scripts and firmware

Contributions welcome — see the relevant folder README for guidance.
