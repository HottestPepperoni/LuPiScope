# Parts list

This folder contains BOMs (Bill of Materials), links to suppliers, recommended alternatives, and notes for the LuPiScope hardware.

Suggested files:

- `parts-bom.md` — human-readable BOM with part numbers and quantities
- `parts.csv` — machine-friendly CSV export of the BOM
- `sources.md` — links to suppliers and price notes

Please include manufacturer, part number, quantity, and a short justification for any recommended alternative.
