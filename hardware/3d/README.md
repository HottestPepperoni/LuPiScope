# 3D models & CAD

This folder contains 3D-printable models, STEP/IGES files, and source CAD files for the LuPiScope enclosure and mechanical parts.

Suggested subfolders:

- `models/` — exported STL/OBJ files for printing
- `cad/` — source CAD files (FreeCAD, Fusion 360, SolidWorks)
- `slices/` — recommended slicer profiles or example G-code

Please include licensing for models and any printing recommendations (material, supports, tolerances).
